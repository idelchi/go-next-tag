```sh

```

### FLAGS THAT GO BRRRRRRR! 🏁

```sh
--version  # FLEXING THE VERSION INFO NO CAP
--bump     # PATCH? MINOR? MAJOR? NONE? YOU CHOOSE KING! (default: patch)

```
